# atividade-poo-1-5 

Descrição do algoritmo 
O algoritmo a seguir tem como objetivo calcular o preço de um combustível, de acordo com o tipo de combustível (gasolina ou álcool) e a quantidade de litros que serão colocados no tanque. O preço é calculado com base em um desconto que é aplicado de acordo com a quantidade de litros colocados.
